Tutorial Dasar

Membuat Akun GitHub

Untuk memulai, kunjungi GitHub dan klik tombol "Sign Up" untuk membuat akun baru. Masukkan informasi yang diperlukan, termasuk email, kata sandi, dan username.

Membuat Repositori Pertama

Setelah masuk, klik tombol "New" di sebelah repositori untuk membuat repositori baru. Beri nama repositori, tambahkan deskripsi (opsional), dan pilih apakah repositori ini akan bersifat publik atau privat.

Mengkloning Repositori ke Komputer Lokal

Untuk mengkloning repositori, buka halaman repositori tersebut di GitHub, lalu salin URL repositori. Buka terminal atau command prompt di komputer lokal, dan ketik perintah:

git clone https://github.com/username/repository.git

Menggunakan Perintah Git Dasar

  • git add: Menambahkan perubahan baru ke staging area.
  • git commit: Menyimpan perubahan di repositori lokal dengan pesan commit.
  • git push: Mengirim perubahan dari repositori lokal ke GitHub.
  • git pull: Mengambil dan menggabungkan perubahan terbaru dari GitHub ke lokal.

Membuat dan Mengelola Cabang

Untuk membuat cabang baru, gunakan perintah:

git branch nama-cabang

Untuk beralih ke cabang tersebut, gunakan perintah:

git checkout nama-cabang
